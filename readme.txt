[27thMEU] Simon
Operation Harvest Red
Eine Kampagne des 27thMEU
Version 1.00

- 2 Fraktionen TvTvAI Mission mit Persistenter Funktion.
- Strategie Element (Zeus) für die Fraktionsführer als möglichkeit der Verbesserten Koordination eigener Truppen
- BLUFOR hat den Auftrag Führer der OPFOR zu finden und Gefangen zu nehmen oder zu Eliminieren.


Missionshinweise

###### 1. Persistence ######
- Vor dem Speichern der Mission müssen sich Grundsätzlich alle Spieler ausßerhalb von Fahrzeugen befinden.
- Spieler die einen Disconnenct hatten, werden automatisch gespeichert. Auch deren letzte Position.
- Die Persistence steuerung kann nur von einem eingeloggten Admin erfolgen. Votet Admin zählen nicht dazu


###### ToDo nach Missionstest ######

- Ausrüstung der Spieler Optimieren (Keine Nachtsicht?)
- Versorgungssystem einbauen (ACE-Logistik? / Script basiert?)
- Performace optimierung erkunden
- Dialoge zur Aufklärungshilfe einbauen
- Kommunikationstürme einbauen (Test der Übertragungsreichweite)
